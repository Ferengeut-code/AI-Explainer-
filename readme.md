AI Explainer Chrome Extension
A Chrome extension that lets you select text on any webpage and get AI-powered explanations via the OpenAI API.

Overview
This extension adds a context menu item that appears when you select text. Clicking it opens a popup where the selected text is automatically loaded. You can then request a short or detailed explanation powered by AI. Users can customize:
-The prompt sent to the AI (e.g., "Explain the word")
-Language of the explanation
-Explanation depth (short or detailed)
-OpenAI API key

Features
-Context menu integration for quick AI explanations
-Customizable prompt, language, and explanation depth
-Persistent settings using Chrome storage API
-Fetches responses from OpenAI Chat Completion API and displays in popup
